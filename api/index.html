Function reference

Gene information

These functions retrieve information summaries, for named genes or genes in a defined genomic range, and displays it as rendered Markdown in the output cell.

gene_info Displays HTML formatted information about one or more human genes.
gene_info_region Displays HTML formatted information about genes in a chromosomal region.

Gene plots

These functions display information graphically together with each either gene structures in a genomic region (gene_plot) or chromosomal ideograms (chrom_ideogram).

gene_plot Plots gene ideograms for a chromosomal region and returns axes for
GenomeIdeogram Class to plot ideograms of chromosomes in a genome assembly.
ChromIdeogram Child class of GenomeIdeogram for plotting single chromosome ideograms.

Gene coordinates

These functions retrieve coordinates of named genes or genes in a chromosomal region.

gene_coords Get gene coordinates for a gene or list of genes.
gene_coords_region Gets gene structure information for genes in a chromosomal region.

ID conversion

These functions converts between NCBI, ENSEMBL and HGCN gene identifiers.

ensembl_id Get ENSEMBL ID for some gene identifier
hgcn_symbol Get HGCN gene symbol for some gene identifier
ensembl2symbol Converts ENSEMBL ID to gene HGCN gene symbol
ensembl2ncbi Converts ENSEMBL ID to gene NCBI ID

STRING networks

These functions retrieve and display STRING network information.

string_network_table Retrieves STRING network for a list of genes and returns it as a pandas.DataFrame.
show_string_network Display STRING network for a list of genes.

GO information

These functions retrieve gene ontology information.

go_info Displays HML formatted information about the given GO terms.
go_term2name Converts a GO term to its name.
go_name2term Converts a GO term name to its term.
get_go_terms_for_genes Get the union of GO terms for a list of genes.
get_terms_for_go_regex Get GO terms for terms matching a regular expression in their description string.
go_annotation_table GO annotations for a given taxonomy id as a pandas.DataFrame.

GO enrichment

These functions allow gene ontology enrichment analyses.

go_enrichment Runs a GO enrichment analysis.
show_go_evidence_codes Display list of GO evidence categories and their codes.
symbols_protein_coding List of protein coding gene symbols for a given taxonomy id.
download_data

show_go_dag_for_gene Displays GO graph for a given gene.
show_go_dag_for_terms Display GO graph for a list of GO terms.
show_go_dag_enrichment_results Displays a GO enrichment analysis results.

Gene annotation

These functions retrieves genes based on their GO annotation.

get_genes_for_go_regex Get gene information for GO terms matching a regular expression in
get_genes_for_go_terms Get gene information for genes with specified GO terms.
gene_annotation_table Gene annotations for a given taxonomy id as a pandas.DataFrame.

horizon Horizon bar plot made allowing multiple chromosomes and multiple samples.
