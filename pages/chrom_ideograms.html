The coordinate system

The x-axis follow the chromosome coordinates along the ideogram. Behind the scenes, the y-axis for each ideogram has the same scale as its x-axis. But this is mapped to a more sensible range that is (0, 10) by default can be controlled by the ylim keyword argument:

g = gplt.ChromIdeogram('chrX', assembly='hg38', ylim=(0, 30))
g.draw_chromosomes()

Ideogram size and plament

g.draw_chromosomes(base=1, height=4)

Genomic segment annotation

g = gplt.GenomeIdeogram(assembly='hg38')

To make sure segments are visible on the plot, the default minimal size of plotteed segments are 200000. This can be changed like this (notice how the chr1 segment is no longer visible):

segments_with_alpha = [(*t, np.random.random()) for t in segments]
segments_with_alpha[:3]
[('chr1', 10065165, 12961789, 0.061182546920045655),
 ('chr1', 15568967, 18751672, 0.021441536887357615),
 ('chr1', 19444467, 23071285, 0.37882307161321216)]
g = gplt.GenomeIdeogram(assembly='hg38') 
g.draw_chromosomes()

Conditional ideogram color

You can also make consequtive segments with individual alpha values to evalutate a statistic across the chromosomes. Below I make 100kb windows with sin values:

length = utils.chrom_lengths['hg38']['chrX']
win = 100_000
x = range(0, length, win)
f = np.linspace(5, 10, len(x))
y = np.sin(2* np.pi * f * x / length) / 2 + 1/2
chrom = ['chrX']*len(x)
shaded_segments = [('chrX', s, s+win, a) for s, a in zip(x, y)]
shaded_segments[:3]
[('chrX', 0, 100000, 0.5),
 ('chrX', 100000, 200000, 0.5100723398623864),
 ('chrX', 200000, 300000, 0.5201534872481727)]
g = gplt.ChromIdeogram('chrX', assembly='hg38') 
g.draw_chromosomes()
g.add_segments(shaded_segments, facecolor='tab:red')

Even better, you can use the add_horizon method to plot a horizon plot on the ideograms:

g = gplt.GenomeIdeogram(assembly='hg38') 
g.draw_chromosomes(base=3, height=3)
g.add_horizon(df, zorder=0)

Context plots

To visualize data in chromosome context, you can plot anything you like above, on top, or below the ideogram:

g = gplt.GenomeIdeogram(assembly='hg38') 
g.draw_chromosomes()
g.map_fun(sns.scatterplot, data=df, x='x', y='y', yaxis=(1, 2), s=5, c='C1', label='sin curve scatter')
g.add_legend()
g = gplt.ChromIdeogram('chrX', assembly='hg38') 
g.draw_chromosomes(base=2)
g.map_fun(sns.lineplot, data=df, x='x', y='y', yaxis=(5, 7), linewidth=2, label='sin curve line')
g.add_legend()

Add gene labels to positions:

g = gplt.GenomeIdeogram(assembly='hg38') 
g.draw_chromosomes()
g.add_labels(sample_gene_coord)

You can also use add_labels to add position markers along a chromosome like this to better see the location

g = gplt.GenomeIdeogram(assembly='hg38', font_size=7) 
g.draw_chromosomes()

ticks = [(c, s, f'{round(s/1000000, 2)}') for c, l in utils.chrom_lengths['hg38'].items() for s in range(0, l, 2_500_000)]
g.add_labels(ticks, base=g.ideogram_base, min_height=g.ideogram_height+1, color='black', zorder=1000)
