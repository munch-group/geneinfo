The coordinate system

The x-axis follow the chromosome coordinates along the ideogram. Behind the scenes, the y-axis for each ideogram has the same scale as its x-axis. But this is mapped to a more sensible range that is (0, 10) by default can be controlled by the ylim keyword argument:

cid = ChromIdeogram('chrX', ylim=(0, 30))
cid.draw_chromosomes()

Showing data on ideograms

gid = GenomeIdeogram() 
gid.draw_chromosomes()
gid.add_segments(segments, facecolor='tab:blue', alpha=0.2)

To make sure segments are visible on the plot, the default minimal size of plotteed segments are 200000. This can be changed like this (notice how the chr1 segment is no longer visible):

segments_with_alpha = [(*t, np.random.random()) for t in segments]
segments_with_alpha[:3]
[('chr1', 10065165, 12961789, 0.18118762093115093),
 ('chr1', 15568967, 18751672, 0.9031709970741433),
 ('chr1', 19444467, 23071285, 0.8957640482957313)]
gid = GenomeIdeogram() 
gid.draw_chromosomes()
gid.add_segments(segments_with_alpha, facecolor='tab:blue')

You can also make consequtive segments with individual alpha values to evalutate a statistic across the chromosomes. Below I make 100kb windows with sin values:

length = utils.chrom_lengths['hg38']['chrX']
win = 100_000
x = range(0, length, win)
f = np.linspace(5, 10, len(x))
y = np.sin(2* np.pi * f * x / length) / 2 + 1/2
chrom = ['chrX']*len(x)
shaded_segments = [('chrX', s, s+win, a) for s, a in zip(x, y)]
shaded_segments[:3]
[('chrX', 0, 100000, 0.5),
 ('chrX', 100000, 200000, 0.5100723398623864),
 ('chrX', 200000, 300000, 0.5201534872481727)]
cid = ChromIdeogram('chrX') 
cid.draw_chromosomes()
cid.add_segments(shaded_segments, facecolor='tab:red')

Even better, you can use the add_horizon method to plot a horizon plot on the ideograms:

gid = GenomeIdeogram() 
gid.draw_chromosomes(base=3, height=3)
gid.add_horizon(df)
gid = GenomeIdeogram() 
gid.draw_chromosomes()
gid.map_fun(sns.scatterplot, data=df, x='x', y='y', yaxis=(1, 2), s=5, c='C1', label='sin curve scatter')
gid.add_legend()
cid = ChromIdeogram('chrX') 
cid.draw_chromosomes()
cid.map_fun(sns.lineplot, data=df, x='x', y='y', yaxis=(1, 2), linewidth=2, label='sin curve line')
cid.add_legend()
